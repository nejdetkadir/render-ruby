# RenderRuby
Ruby bindings for [Render API](https://api-docs.render.com)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'render_ruby'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install render_ruby
